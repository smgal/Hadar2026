# 키 입력 정책 (Key Input Policy)

본 프로젝트는 고전 '또 다른 지식의 성전(Hadar)'의 조작감을 유지하면서 현대적인 PC 게임 환경에 맞춘 키 입력을 지원합니다.

## 1. 기본 조작키

### 이동 (Movement)
*   **화살표 키 (↑, ↓, ←, →)**
*   **WASD (W, A, S, D)**

### 확인 및 액션 (Confirm / Action)
*   **Enter**
*   **E**
*   용도: 대화 진행, 문 열기, 표지판 읽기, 메뉴 항목 선택 등.

### 메인 메뉴 및 취소 (Main Menu / Cancel)
*   **Esc**
*   **Q**
*   **Space** (PC 전용 메인 메뉴 호출)
*   용도: 필드에서 메인 메뉴 열기, 열린 메뉴나 윈도우 닫기(취소), 대화 넘기기 등.

## 2. 모드별 상세 동작

| 모드 | 이동 | 확인/액션 (Enter, E) | 메뉴/취소 (Esc, Q, Space) |
| :--- | :--- | :--- | :--- |
| **필드 (Map)** | 캐릭터 이동 | 상호작용 (대화, 조사 등) | 메인 메뉴 열기 |
| **대화 (Dialogue)** | (기능 없음) | 다음 대화로 넘어가기 | 다음 대화로 넘어가기 |
| **메뉴 (Menu)** | 커서 이동 | 선택 항목 확정 | 메뉴 닫기 또는 상위 단계로 |
| **윈도우 (Window)** | 커서 이동 | 현재 창의 동작 확정 | 창 닫기 (Cancel) |
